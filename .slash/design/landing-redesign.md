# Landing Page Redesign

## Core Concept
The landing page should feel like an entry point to the creative studio. Dark, immersive, with a clear focus on the output (the logos).

## Section-by-Section Specs

### 1. Hero Section
- **Layout**: Centered text, large and confident.
- **Headline**: "AI와 대화하며 나만의 로고를 완성하세요"
- **Subheadline**: "복잡한 디자인 툴 없이, 채팅만으로 전문가 수준의 로고를 만들어냅니다."
- **CTAs**: 
  - Logged Out: "무료로 시작하기" (Primary)
  - Logged In: "대시보드로 이동" (Primary)
- **Visual**: A mock UI or an animated grid of high-quality generated logos directly below the hero text, fading into the background at the bottom.

### 2. How It Works (작동 방식)
- **Layout**: 3-column grid.
- **Cards**: 
  1. **인터뷰**: AI가 브랜드의 핵심 가치를 묻습니다.
  2. **생성**: 몇 초 만에 다양한 시안을 제안합니다.
  3. **수정 및 내보내기**: 피드백을 반영하고 고해상도로 다운로드합니다.
- **Style**: Minimal cards with subtle icons (perhaps using simple geometric shapes or green accents).

### 3. Features (기능)
- **Layout**: 2x3 grid.
- **Style**: Cards with `--bg-secondary`, highlighting specific technical or creative features (e.g., "무제한 시안", "배경 제거", "SVG 변환").
- **Hover**: Cards lift slightly via border color change (to `--border-secondary`).

### 4. Pricing Teaser
- **Layout**: 3 cards, middle card (Pro) highlighted.
- **Link**: "요금제 전체 보기" leading to `/pricing`.

### 5. Final CTA
- Centered, mirroring the hero but simpler.
- "지금 바로 당신의 브랜드를 시각화하세요."
